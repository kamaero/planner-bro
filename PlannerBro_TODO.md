## P0 (текущий фокус, до следующего релиза)

   - Общий стиль для email и web-preview.

   - «Открыть задачу», «Взять в работу», «Отметить check-in», «Эскалировать».
   - Безопасные deeplink/action endpoints с проверкой прав.

4. 🚧 Контроль доставки
   - Статусы отправки: queued/sent/delivered/failed/skipped.
   - Повторные попытки + причины отказов.

5. 🚧 Telegram UX
   - Ролевые дайджесты в Telegram.
   - Компактный и полный режимы сообщений.
   - Инлайн-кнопки быстрых действий.

6. 🚧 Надёжность рассылок
   - Очереди, ретраи, идемпотентность задач.
   - Защита от дублей при рестартах/повторной доставке.

## Критерии готовности ближайшего релиза

1. Все ролевые дайджесты отправляются корректно (email + Telegram) минимум 7 дней без ручных правок.
2. HTML-отчёты выглядят одинаково в ключевых клиентах (Gmail, Outlook, web preview).
3. Быстрые действия из отчётов безопасны и проходят по правам.
4. Есть прозрачный контроль доставки и ошибок в UI.
5. Анти-шум уменьшает повторные уведомления без потери критичных событий.
